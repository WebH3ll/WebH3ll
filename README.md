    - ex) 사용자로부터 ip를 입력받으면 서버에서 ping을 날려주는 서비스
    - ex) 사용자로부터 url을 입력받으면 서버에서 dnslookup을 해주는 서비스
- 회원가입, 로그인, 로그아웃 기능 구현

    - 그 외에 자신의 사이트에서 제한된 정보를 볼 수 있는 관리자 권한 구현
- 게시판 기능
    - 게시글 작성
    - 게시글 읽기
    - 게시글 삭제 및 수정
    - 비밀 게시글
- 파일 업로드 및 다운로드 기능
